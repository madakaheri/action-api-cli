# サポート

## 問い合わせ方法

このテンプレートパッケージの利用で支援が必要な場合は、次を確認してください。

- [README.md](../README.md) の内容を確認する
- 不具合や機能要望は GitHub Issue を作成する
- 問題を報告する際は再現手順と利用環境を記載する

## Issue 作成時のお願い

Issue を作成する際は、できるだけ次の内容を含めてください。

- 期待していた動作
- 実際の動作
- 再現手順
- Node.js のバージョンと関連する環境情報

## コントリビューション

改善への参加方法は [CONTRIBUTING.md](CONTRIBUTING.md) を参照してください。
