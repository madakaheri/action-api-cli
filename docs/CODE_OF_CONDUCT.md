# 行動規範

## 基本方針

このプロジェクトでは、誰もが安心して参加できる礼節ある環境を保つことを期待します。

望ましい行動の例:

- 議論やレビューで相手を尊重する
- 建設的なフィードバックを受け入れる
- 個人よりもプロジェクト全体にとって最善の判断を優先する
- 他の参加者に配慮し、共感を持って接する

許容されない行動の例:

- 嫌がらせや差別的な発言
- 個人攻撃や荒らし行為
- 許可のない個人情報の公開
- そのほか、専門的または健全な共同作業の場にふさわしくない行為

## 対応方針

プロジェクトのメンテナーは本規範の解釈と適用に責任を持ち、違反するコメント、コミット、Issue、プルリクエストに対して修正、削除、却下などの対応を行う場合があります。

## 報告方法

不適切な行為を受けた、または確認した場合は、公開 Issue を作成する前に非公開の手段でメンテナーへ連絡してください。

## 適用範囲

この行動規範は、プロジェクトに関連する場所、およびプロジェクトやコミュニティの一員として行動している公的な場に適用されます。
