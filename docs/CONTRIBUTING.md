# コントリビューションガイド

このパッケージテンプレートへの貢献を歓迎します。

## 開発環境の準備

1. 依存関係をインストールします。
2. 必要な変更を行います。
3. プルリクエストを作成する前に各種チェックを実行します。

```sh
npm install
npm test
```

## プロジェクト構成

- `src/`: パッケージのソースコード
- `@types/`: `npm run build` で生成される型定義ファイル

## プルリクエスト

プルリクエストは目的を絞り、次の内容を含めてください。

- 変更内容の明確な要約
- 挙動が変わる場合はテストまたは確認手順
- 公開される挙動が変わる場合は関連ドキュメントの更新

## ビルド

型定義を生成するにはビルドスクリプトを実行してください。

```sh
npm run build
```
